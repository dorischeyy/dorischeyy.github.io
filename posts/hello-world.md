# 第一篇文章：开始写博客

很久以前就想建一个博客，一直没动手。今天终于做了。

## 为什么要写

不是为了影响别人，也不是为了流量。只是觉得，有些东西如果不写下来，就会消失。

想法是流动的，感受是短暂的。文字是一种留存的方式。

## 打算写什么

没有严格计划。可能是：

- 读了一本书之后的某些感受
- 工作中遇到的有趣问题
- 某个下午突然想清楚的事情
- 一些日常的记录

不追求系统性，也不强迫自己定期更新。想写的时候就写。

## 关于这个博客

用纯静态 HTML 搭建，部署在 GitHub Pages 上。没有数据库，没有后端，文章以 Markdown 文件存放。

简单、干净、够用。

---

开始了。
